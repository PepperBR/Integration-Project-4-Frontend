# Instruções de Utilização — Projeto_Completo

## Execução do Projeto

1. Execute o arquivo **`Projeto_Completo.exe`**
   - Nessa etapa será gerado os seguintes executáveis:
     - `Projeto_GRPC.exe`
     - `Display.exe`

2. Execute o **`Projeto_GRPC.exe`**
   - Surgirá um terminal que indica que o server está funcionando.

3. Em seguida, execute o **`Display.exe`**
   - A interface gráfica será iniciada.
   - Ela se conectará ao servidor que está rodando no em paralelo no **`Projeto_GRPC.exe`**
  
4. Quando desejar parar a execução do projeto completo, basta fechar as janelas do **`Display.exe`** e **`Projeto_GRPC.exe`**.

---

## Observações

- Caso apareça um terminal quando executar o **`Projeto_Completo.exe`**, escreva **`Y`** quando pedir uma resposta.
- O `Projeto_GRPC.exe` deve ser iniciado antes do `Display.exe` e deve ser mantido aberto durante toda a execução.
