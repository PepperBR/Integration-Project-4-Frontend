# Instruções de Utilização — Projeto_Completo

## Execução do Projeto

1. Execute o arquivo **`Projeto_Completo.exe`**
   - Essa etapa irá gerar os executáveis:
     - `Projeto_GRPC.exe`
     - `Display.exe`

2. Abra o **`Projeto_GRPC.exe`**
   - O backend do sistema será iniciado.

3. Em seguida, abra o **`Display.exe`**
   - A interface gráfica será iniciada e conectada ao backend.

---

## Observações

- Caso apareça um terminal quando executar o **`Projeto_Completo.exe`**, escreva **`Y`** quando pedir uma resposta.
- O `Projeto_GRPC.exe` deve ser iniciado antes do `Display.exe`.
